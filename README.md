# NekoCore 

> **Разработано с душой и благими намерениями.**

Этот проект — попытка создать удобную, свободную и красивую платформу для общения. 
⚠️ **Важное примечание:** Проект **реализован не до конца**. Это "живой" код, в котором могут встречаться баги, недописанные функции или места, требующие оптимизации. Это фундамент, который можно развивать дальше.

## Возможности (на данный момент)

* **Real-time чат:** Мгновенный обмен сообщениями (текст, картинки, аудио, видео-кружочки).
* **Голосовая связь:** Комнаты и звонки (интеграция с AgoraRTC).
* **Кастомизация:** Мощный редактор тем, настройки профиля, аватарок и баннеров.
* **Админ-панель:** Система ролей (User, Admin, Super/Root), бан-система, модерация.
* **PWA:** Можно установить как приложение на телефон или ПК.
* **Безопасность:** Базовые проверки приватности и черные списки.

## Установка и Запуск

Так как проект использует современные веб-технологии (Service Workers, Modules) и требует доступа к микрофону/камере, просто открыть `index.html` **недостаточно**.

### Требования
1.  Аккаунт **Firebase** (Google).
2.  Аккаунт **Agora.io** (для голосовой связи).
3.  Локальный веб-сервер (или хостинг с HTTPS).

### Шаг 1: Скачивание
Скачайте все файлы проекта и сохраните структуру папок

Шаг 2: Настройка Базы Данных (Firebase)

    Зайдите на Firebase Console.

    Создайте новый проект.

    В разделе Authentication включите метод входа Email/Password.

    В разделе Realtime Database создайте базу и выберите правила (для теста можно поставить read: true, write: true.

    Перейдите в настройки проекта -> General -> Your apps -> Web app (</>). Скопируйте конфиг.

Откройте файл functions/config.js и замените заглушки на ваши данные:

const CFG = {
    fb: {
        apiKey: "ВАШ_API_KEY",
        authDomain: "ВАШ_PROJECT.firebaseapp.com",
        databaseURL: "ВАШ_URL_БАЗЫ_ДАННЫХ",
        projectId: "ВАШ_PROJECT_ID",
        storageBucket: "ВАШ_BUCKET",
        messagingSenderId: "ВАШ_SENDER_ID",
        appId: "ВАШ_APP_ID"
    }
};

Шаг 3: Настройка Голосовой Связи (Agora)

    Зайдите на Agora.io.

    Создайте проект и получите App ID (режим Testing без сертификата).

    Откройте файл functions/voice.js (в самом верху) и вставьте свой ID:

JavaScript

const AGORA_APP_ID = "ВАШ_AGORA_APP_ID"; 

Шаг 4: Запуск

Из-за политики безопасности браузеров (CORS и доступ к медиа-устройствам), проект нужно запускать через локальный сервер.

Вариант А (VS Code): Установите расширение Live Server, нажмите правой кнопкой на index.html -> "Open with Live Server".

Вариант Б (Хостинг): Загрузите файлы на GitHub Pages, Vercel или любой другой хостинг. Важно: Для работы микрофона и камеры сайт должен открываться по протоколу HTTPS.
